#### Codédex Python Checkpoints

A collection of three terminal-based Python projects built to practice fundamental concepts like variables, control flow, and loops.

##### Projects

- **Area Calculator** (`calculator.py`)
  A command-line tool that calculates the area of basic geometric shapes (Square, Rectangle, Triangle, Circle) based on user input.

- **Rock Paper Scissors Lizard Spock** (`rock_paper_scissors.py`)
  An extended version of the classic game played against the CPU, implementing complex conditional logic.

- **Terminal Adventure** (`terminal_game.py`)
  A text-based interactive adventure game featuring multiple paths and narrative choices.
